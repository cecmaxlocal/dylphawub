# dylphawub

![IDEWhatzapp](./matrix/cec/images/logon.png)

## Overview
**dylphawub** is a project developed in the **Raku programming language**, focused on creating an integrated experience for WhatsApp calls via TV and desktop platforms. This repository contains development files, server code, and automation scripts aimed at enabling call integration and management using TV and desktop interfaces.

## Features
- WhatsApp call integration accessible through TV and desktop.
- Server-side components written in Raku to manage call sessions and routing.
- Modular design enabling easy extension and maintenance.
- Support for desktop notifications and interaction hooks.
- Tools to streamline development and deployment within a Raku environment.

## Development Environment
Recommended IDEs and editors for working on this project:
- Visual Studio Code with Raku plugins.
- Comma IDE - a dedicated Raku development environment.
- Vim or Emacs with Raku syntax support.
- Use of Rakudo compiler for building and testing.

## Project Structure
- `./dylphawub` - Main source files and modules.
- `./scripts` - Utility and automation scripts.
- `./tests` - Test suite for validating components.
- `./docs` - Documentation and guides.

## How to Run
1. Install Rakudo Raku compiler.
2. Clone the repository.
3. Use terminal commands or IDE run configurations to start the Raku server or scripts.
4. Access the application via supported desktop or TV interfaces.
